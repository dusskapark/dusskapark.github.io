목적

본 문서는 박주형의 다양한 역량을 그 고용주의 요청(http)으로 활용할 수 있는 API에 대해 설명하며, 박주형이 제공하는 각각의 기능별 상세 설명을 포함합니다.

API 종류

account_circle 프로필 API

박주형이라는 개인에 대한 간단한 정보를 출력하는 API입니다.

assessment 데이터 분석 API

Google Analytics와 R을 활용한 데이터 수집, 설계, 분석 능력을 호출하는 API 입니다.

sms 서비스 운영 API

각종 VOC와 장애 상황에 빠르게 대응하고, 문제의 원인을 찾아 다양한 정책들을 수립/검토하는 능력을 호출하는 API 입니다.

mms 서비스 기획 API

최신 기술이나 서비스 트랜드, 고객 피드백 그리고 데이터 분석을 기반으로 의사결정을 내리고, 이를 논리적으로 기획서와 서비스에 반영하는 능력을 호출하는 API 입니다.
