# MCP Magic retrospective research archive

이 폴더는 [MCP Magic 앱 회고](/blog/mcp-magic-retrospective/)에서 사용한 공개 가능한 근거 자료를 모아 둔 아카이브다.

## 구성

- `ga4/`: 2025-07-01 ~ 2026-06-30 GA4 집계, 분석 문서, 정적 리포트
- `stores/`: App Store Connect, Microsoft Store, GitHub Release의 다운로드·설치 집계
- `aptabase/public-session-summary.json`: 2026-02-01 이후 Aptabase 세션 분석의 익명 집계 요약

## 공개 범위

세션 ID, 익명 사용자 ID, 원본 이벤트 행, 지역별 세션 원본 등은 공개 저장소에 옮기지 않았다. 본문과 공개 자료에는 집계된 사용 규모와 작업 깊이만 남겼다.

다운로드·설치 수는 고유 사용자 수가 아니라 배포 채널별 집계다. 같은 사용자의 재다운로드, 업데이트 또는 여러 배포 채널 사용이 중복으로 포함될 수 있다.
